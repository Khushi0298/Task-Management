# Task-Management
A Task Management System that enables users to efficiently organize, track, and update their daily tasks through a secure and intuitive interface. The backend is powered by Spring Boot with JWT-based authentication, ensuring safe access to all task operations including creation, updating, deletion, searching, and progress tracking.
